DevOps Life Cycle

The steps involved in DevOps Lifecycle are:


Plan
This step involves basic planning of what to do and how to do.
Develop
This step involves developing the basic code that is required for project.
Build
This step involves building the software and hardware componenets required for the project.
Test
This step involves testing the built software using various sources.
Release
This step involves releasing the tested software if it has no bugs in it.
Deploy
This step involves deploying the sotware or project in the client system.
Operate
This step involves operating the software or project in the client environment.
Monitor
This step involves Monitoring after publishing.
